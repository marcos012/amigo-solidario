## Stack

- [NodeJS](https://nodejs.org/en/)
- [Express](https://expressjs.com/pt-br/)
- [React](https://github.com/facebook/react)

## Iniciar aplicação
**Pré-requisitos:** <br>

[Node/NPM](https://nodejs.org/en/)

**Backend:**

Intalar as dependências
```
npm i
```
Rodar as migrations do projeto
```
npm run db:latest
```
Start na aplicação
```
npm run dev
```
<br />

**Frontend:**

Intalar as dependências
```
npm i
```
Start na aplicação
```
npm start
```
<br />
